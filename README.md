# Recipes

A personal collection of recipes stored in markdown format for easy organization and sharing.

## Structure

All recipes are stored in the `recipes/` directory. You can organize them however you prefer - by cuisine, meal type, or any other system that works for you.

## Recipe Format

Each recipe follows a standard markdown template that includes:
- Title and description
- Prep time, cook time, and total time
- Servings
- Ingredients list
- Step-by-step instructions
- Notes and tips
- Tags for easy searching

## Usage

1. Navigate to the `recipes/` directory
2. Create a new markdown file for your recipe
3. Use the `recipe-template.md` file in the root directory as a starting point
4. Add tags to help with organization and searching

## Contributing

Feel free to add your own recipes! Just follow the established format and place them in the `recipes/` directory.
